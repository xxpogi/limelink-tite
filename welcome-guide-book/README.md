# Welcome Guide Book

Adds a `/guide` command with practical survival tips for new players.

## Features
- `/guide` prints a short beginner-friendly checklist.
- Sends a reminder on first join.

## Requirements
- Paper or Folia 1.20+
- Skript 2.7+

## Installation
1. Copy `welcome-guide-book.sk` into `plugins/Skript/scripts/`.
2. Run `/skript reload welcome-guide-book`.
