# Survival Idea Board

Provides random survival project ideas so players always have a next goal.

## Features
- `/idea` shows one random survival project idea.
- `/idea list` shows all configured ideas.
- `/ideaadd <text>` lets admins add ideas live.
- Sends one random suggestion on join.

## Requirements
- Paper or Folia 1.20+
- Skript 2.7+

## Installation
1. Copy `survival-idea-board.sk` into `plugins/Skript/scripts/`.
2. Run `/skript reload survival-idea-board`.

## Permissions
- `idea.admin` — use `/ideaadd`.
